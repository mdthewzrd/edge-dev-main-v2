1. **Create the Project**:

   - Click "New Project"
   - Select "Board" (Kanban)
   - Name: "RENATA V2 Build"
   - Repository: Select "ce-hub"
   - Click "Create"

2. **Create Columns**:
   - **Backlog** (default)
   - **In Progress**
   - **Review**

   - Copy tasks from this document

   - Assign labels and milestones

   - Set due dates
